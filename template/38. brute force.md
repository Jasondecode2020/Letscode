### 1316. Distinct Echo Substrings

```python
class Solution:
    def distinctEchoSubstrings(self, text: str) -> int:
        n, visited, res = len(text), set(), 0
        for i in range(n):
            for j in range(i + 1, n):
                if j * 2 - i <= n and text[i:j] == text[j:j*2-i] and text[i:j] not in visited:
                    res += 1
                    visited.add(text[i:j])
        return res
```
