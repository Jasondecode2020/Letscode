### 13. Roman to Integer

```python
class Solution:
    def romanToInt(self, s: str) -> int:
        d = {
            'I': 1,
            'V': 5,
            'X': 10,
            'L': 50,
            'C': 100,
            'D': 500,
            'M': 1000
        }
        
        res = d[s[0]]
        for i in range(1, len(s)):
            res += d[s[i]]
            if d[s[i]] > d[s[i - 1]]:
                res -= 2 * d[s[i - 1]]
        return res
```
